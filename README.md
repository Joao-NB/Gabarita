# Gabarita
Gabarita é o aplicativo de quizzes inteligente que ajuda estudantes a aprender e revisar matérias de forma rápida e divertida. Escolha a matéria e o assunto, e receba questões geradas automaticamente pela IA para testar seus conhecimentos!  Tecnologias usadas: Angular, TypeScript, TailwindCSS, Node.js, Express.js, MongoDB e Google Gemini AI.
